### Interactive CLI Experience

- Target tool selection: MultiSelect prompt when `--target` is not specified (no default selection)
- Install scope selection: Confirmation prompt for project-local installation with full path display
- Installation preview: Shows all target paths before execution with visible colors
- Unified confirmation: Final confirmation before install/remove operations

### Architecture Improvements
- **ToolProvider Interface**: Abstracted tool-specific behavior into `ToolProvider` interface
- **ToolType Enum**: Added type-safe enumeration for tool types (`ToolGemini`, `ToolClaude`, `ToolCodex`)
- **Provider Registry**: Centralized provider management with `GetProvider()` and `ParseProviders()`
