# v0.7.0 - Extended Provider Support

Release Date: 2026-01-15

## New Features

### Cline IDE Support
- Added `cline` provider for Cline IDE integration
- Global skills directory: `~/.cline/skills/`
- Project-local skills: `.cline/skills/`
- Usage: `skillsync install repo --target cline`

### Goose AI Support
- Added `goose` provider for Goose AI integration
- Global skills directory: `~/.config/goose/skills/`
- Project-local skills: `.goose/skills/`
- Usage: `skillsync install repo --target goose`

### Crush Support
- Added `crush` provider for Charmbracelet Crush terminal tool
- Global skills directory: `~/.config/crush/skills/`
- Project-local skills: `.crush/skills/`
- Usage: `skillsync install repo --target crush`

### Droid (Factory AI) Support
- Added `droid` provider for Factory AI's Droid
- Global skills directory: `~/.factory/skills/`
- Project-local skills: `.factory/skills/`
- Usage: `skillsync install repo --target droid`

### Kilo Code Support
- Added `kilocode` provider for Kilo Code VSCode extension
- Global skills directory: `~/.kilocode/skills/`
- Project-local skills: `.kilocode/skills/`
- Usage: `skillsync install repo --target kilocode`

### Roo Code Support
- Added `roocode` provider for Roo Code VSCode extension
- Global skills directory: `~/.roo/skills/`

- Usage: `skillsync install repo --target roocode`

### VSCode (Copilot) Alias
- Added `vscode` as an alias for `copilot` provider
- Shares the same directory configuration as GitHub Copilot
- Usage: `skillsync install repo --target vscode`
