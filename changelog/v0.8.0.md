# v0.8.0 - Self-Update & One-Liner Installation

Release Date: 2026-01-15

## New Features

### One-Liner Installation
- Support installing SkillSync and skills in a single command
- Usage: `curl ... | bash -s -- install <repo>`
- Supports all `skillsync install` flags

```bash
# Install SkillSync + target skill repository in one command
curl -fsSL https://raw.githubusercontent.com/AlfonsSkills/SkillSync/main/install.sh | bash -s -- install AlfonsSkills/skills

# With target tool specification
curl -fsSL https://raw.githubusercontent.com/AlfonsSkills/SkillSync/main/install.sh | bash -s -- install AlfonsSkills/skills -t claude,gemini
```

### Auto-Update Command
- New `upgrade` command for self-updating SkillSync
- Checks GitHub Releases for latest version
- Downloads and replaces binary automatically

```bash
# Check for updates
skillsync upgrade --check

# Upgrade to latest version
skillsync upgrade
```

### Enhanced Version Information
- Added author info to version output
- Added project URL to version output
- Cleaner version format using exact git tags

```
SkillSync v0.8.0
Git Commit: b8a5a0d
Build Time: 2026-01-15_06:49:05

Author:  Alfons <alfonsxh@gmail.com>
Project: https://github.com/AlfonsSkills/SkillSync
```

### Smart Update Check
- Automatically checks for updates after `install` command
- Utilizes existing network connection (no extra latency)
- 3-second timeout to avoid blocking
- Skips check for dev builds

## Changes

- Makefile now uses exact git tags instead of `describe` format
- Version displays clean `v0.8.0` instead of `v0.7.0-N-gHASH`
- Update check only triggers on `install`, not all commands
